# 项目经历

#### 介绍
本科在校期间编写的代码整理汇总，主要偏向算法实现与优化，包含数据分析、机器学习等方面内容。

希望各位不吝赐教。

#### 软件架构
软件架构说明

代码实现偏理论，因此比较偏算法底层，实际上有更好的第三方库实现。主要是为了加深对算法的理解并提高作者的编程能力。

机器学习的框架基于原始论文复现，如LeNet、AnoGAN等。

#### 使用说明
包含以下几种语言：

1.  python
2.  C++
3.  MySQL
4.  Matlab

不需要特别的运行环境，但是版本问题是不可避免的。如果运行出错，请检查软件版本
